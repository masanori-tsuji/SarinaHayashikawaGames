●ストーリー●

のんびり気ままに宇宙旅をしていたドラゴン型宇宙人ネルヴァスだったが、ある日突然
「ちょっと遠くまで乗せていってほしい」と鳥型宇宙人オリオットが乗り込んできた！
しかも宇宙船を戦闘機仕様に改造されてしまう・・・
とんだ災難だと思いつつも悪い奴ではなさそうと思うネルヴァス・・・仕方なく目的地まで乗せてあげることにしたのであった。

●ゲーム内説明●

〇スコア
・敵を倒したり、ボムを使って弾を吹き飛ばしたりすると加算されます。
　グレイズボーナスや残機ボーナスでも加算されます。
　ランキング１位を目指そう。

〇グレイズ
・弾にかすると加算されます。
ギリギリを攻めれば攻めるほどボーナス加点されます。

〇ボム
・画面内の弾を消します。エネミーの体力も削れるので効果的に使おう。
　一度死ぬとリセットされます。

〇ライフ
・０になるとゲームオーバーです。

〇パワー
・ためると自機の弾が強化されます。
　一度死ぬと減るので注意！
　４段階あります。

〇コンティニュー
　3回までできます。
　ライフ、ボム共にリセットします。
　ノーコンティニュークリアでエンディングが変わります。


※ランキングについて
お手数をおかけしますが、同封のserver.exeを同時に起動してプレイをお願いします。
